# 项目流程记录
1. 创建项目
   ```js
      pnpm create vite my-vue-app
   ```
2. 安装依赖
### 1. 添加路由
   ```js
      pnpm i vue-router@4
   ```
### 2. 安装pinia
   ```js
      pnpm i pinia
   ```
### 3. 安装axios
   ```js
      pnpm i axios -S
   ```
### 4. 安装Echarts
   ```js
      pnpm i echarts -S
   ```
### 5. 安装ui组件库
   ```js
      pnpm i element-plus -S
   ```
### 7. 安装dayjs
   ```js
      pnpm i dayjs -S
   ```
### 8. 安装moment
   ```js
      pnpm i moment -S
   ```
### 9. 安装normalize.css
   ```js
      pnpm i normalize.css -S
   ```
